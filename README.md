# Marble Roulette - Premium Physics Race

A high-performance, physics-based marble race simulation built with Vite and Matter.js.

## 특징 (Features)
- **최고급 디자인**: 네온 다크 모드와 글래스모피즘 UI.
- **방대한 맵**: 9,500px의 거대한 맵과 10가지 이상의 기발한 장애물 구역.
- **당첨자 맞춤 설정**: 1명부터 20명까지 자유로운 당첨 인원 설정 및 자동 레이스 종료.
- **결과 저장**: 레이스 결과를 실시간으로 내보내고 `.txt` 파일로 다운로드 가능.
- **스마트 카메라**: 1등 마블을 실시간으로 추적하는 부드러운 카메라 시스템.

## 설치 및 실행 (Setup & Run)
1. **Node.js**가 설치되어 있어야 합니다.
2. 프로젝트 폴더에서 터미널을 열고 다음 명령어를 실행하세요:
   ```bash
   npm install
   npm run dev
   ```
3. 브라우저에서 `http://localhost:5173` 접속.
