# Marble Roulette 공유 및 다른 PC 사용 가이드

이 프로젝트는 웹 기반 애플리케이션으로, 다른 PC에서도 쉽게 실행하고 공유할 수 있습니다. 

## 1. 간단한 공유 방법 (빌드 버전 활용)
이미 프로젝트 폴더 내 `dist` 폴더에 실행 가능한 파일들이 만들어져 있습니다.

1. `dist` 폴더를 다른 PC로 복사합니다.
2. **주의**: 보안 정책상 `index.html`을 단순히 더블 클릭하면 작동하지 않을 수 있습니다.
3. 가장 쉬운 해결책은 `npx serve dist` 명령어를 사용하거나, 간단한 웹 서버(Live Server 등)를 사용하는 것입니다.

## 2. 개발 환경 통째로 옮기기 (추천)
다른 PC에도 Node.js가 설치되어 있다면 가장 안정적인 방법입니다.

1. `marble-roulette` 폴더 전체를 복사합니다. (`node_modules` 폴더는 제외해도 됩니다)
2. 새로운 PC에서 터미널을 열고 프로젝트 폴더로 이동합니다.
3. 다음 명령어를 실행합니다:
   ```bash
   npm install
   npm run dev
   ```

## 3. 웹에 올려서 링크로 공유하기 (최고의 방법)
어디서든 링크만 있으면 접속할 수 있게 만드는 방법입니다.

*   **GitHub Pages**: 무료로 호스팅할 수 있습니다.
*   **Netlify / Vercel**: `dist` 폴더를 드래그 앤 드롭하는 것만으로 1분 만에 웹 사이트를 만들 수 있습니다.
    *   [Vercel](https://vercel.com/) 에 접속하여 회원가입 후 `dist` 폴더를 업로드하세요.

## 4. 파일 한 개로 만들기
Vite 플러그인(`vite-plugin-singlefile`)을 사용하면 HTML 파일 단 하나에 모든 코드(JS, CSS)를 담을 수 있습니다. 필요하시면 이 설정을 도와드릴 수 있습니다.
